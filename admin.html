Banner de inicio
Imagen de fondo de la sección hero
Vista previa actual
Sin imagen de banner — se usa el fondo degradado por defecto
¿Cómo funciona?
El banner se guarda en el navegador (localStorage). Si querés que todos los visitantes lo vean, necesitás editar el archivo index.html directamente con el link de la imagen.

Formato recomendado: imagen apaisada (1920×600px), oscura o con texto blanco encima. Funciona mejor con imágenes de Google Drive convertidas.
Subir imagen desde tu computadora
Requiere estar conectado a Google Drive. La imagen se sube automáticamente y se guarda como banner.
O pegá un link de Google Drive convertido
El banner guardado se muestra en index.html si tenés el código de integración activo (ver más abajo).
Código a agregar en index.html
Para que el banner funcione en el sitio público, pegá este código al final del <script> principal de tu index.html, antes del cierre </script>:
// Cargar banner desde admin
(function(){
  const bUrl = localStorage.getItem('founder_banner_url');
  if(bUrl){
    const hero = document.querySelector('.hero');
    if(hero){
      const img = document.createElement('img');
      img.src = bUrl;
      img.style.cssText='position:absolute;inset:0;width:100%;height:100%;object-fit:cover;opacity:0.35;pointer-events:none;';
      hero.style.position='relative';
      hero.insertBefore(img, hero.firstChild);
    }
  }
})();
